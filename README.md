# 氢原子电子云可视化

## 项目简介
本项目通过基于真实波函数的渲染，提供了氢原子电子云的可视化工具。用户可以通过交互式界面探索电子云的概率分布特性，支持多种模式和功能。

## 功能概述

### 1. 模式选择
- **默认模式**：仅允许选择一个轨道进行渲染。
- **多选模式**：允许选择多个轨道叠加渲染。
- **比照模式**：最多选择三个轨道，分别用红、绿、蓝三种颜色渲染。

### 2. 坐标轴功能
- **滑动条控制**：通过滑动条调整坐标轴大小，比例系数范围为 `0-1.05`。
  - 比例系数 × 轨道半径 = 坐标轴大小。
  - 滑动条值为 `0` 时隐藏坐标轴。
- **状态控制**：
  - 渲染前：滑动条可修改。
  - 渲染中：滑动条置灰不可修改。
  - 渲染后：滑动条恢复可修改。
  - 暂停时：滑动条可修改。
- **实时更新**：采样过程中，坐标轴大小会根据最远采样点动态调整。

### 3. 数据面板
- **自动弹出**：采样完成后，数据面板会自动弹出。
- **图表切换**：支持径向分布和角向分布图表的切换。
- **对比模式**：在比照模式下，显示不同轨道的概率密度对比图。

### 4. 其他功能
- **渲染相位**：显示轨道的正负相位（红色和蓝色）。
- **3D角向形状**：基于采样数据生成角向分布的三角网格曲面。
- **实时调整**：支持动态调整点大小、不透明度和生成速度。

---

## 使用说明

### 1. 基本操作
1. 选择轨道和模式。
2. 调整参数（如坐标轴大小、点大小、不透明度等）。
3. 点击“启动”开始采样。
4. 采样完成后，可查看数据面板中的图表。

### 2. 状态控制
- **启动**：点击“启动”后，保留用户设置的坐标轴比例系数。
- **暂停**：点击“暂停”可临时停止采样。
- **重置**：点击“重置”清空所有设置和渲染结果。

---

## 技术实现

### 1. 渲染技术
- 基于 `Three.js` 实现硬件加速的 3D 渲染。
- 使用 `BufferGeometry` 和 `PointsMaterial` 渲染大量采样点。

### 2. 数据处理
- 径向分布和角向分布基于采样数据实时计算。
- 使用 `Chart.js` 绘制概率密度图表。

### 3. 状态管理
- 通过全局状态对象管理采样和渲染状态。
- 提供两种重置方式：
  - `resetState()`：完全重置所有状态。
  - `resetSamplingState()`：保留用户设置的采样重置。

## 注意事项
1. **性能优化**：
   - 对于低配置设备，建议将最大采样点数设置为 `24000` 以下。
   - 复杂轨道（如 `4f`）需要更高采样点数以获得清晰效果。
2. **浏览器兼容性**：
   - 推荐使用支持 WebGL 2.0 的现代浏览器（如 Chrome、Firefox）。
3. **数据准确性**：
   - 采样结果具有统计意义，单次运行可能与理论值存在偏差。
   - 增加采样点数可减小误差。
